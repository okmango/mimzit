Create `Mimzit/Features/Recording/RecordingViewModel.swift`:

`@Observable @MainActor final class RecordingViewModel: NSObject, AVCaptureFileOutputRecordingDelegate`:

**Properties:**
- `let content: ReferenceContent` — injected at init
- `let playbackEngine = PlaybackEngine()`
- `let captureEngine = CaptureEngine()`
- `var isRecording = false`
- `var videoBlend: Float = 0.5` (D-08: default center blend)
- `var audioBlend: Float = 0.0` (D-08: default reference-only; maps to `playbackEngine.volume = 1.0 - audioBlend`)
- `var activeViewMode: ViewMode = .blend` (D-10: default blend mode)
- `var controlsVisible = true`
- `var isFaderDragging = false` — prevents auto-hide while dragging (D-04)
- `var syncTimestamp: Double = 0` (REC-05)
- `var lastRecordingURL: URL?`
- `var teleprompterScrolling = false` — driven by isRecording for teleprompter (D-13)
- `var scrollSpeed: Double = 1.0` — teleprompter speed 1x-4x
- `var textOverlayOpacity: Float = 1.0` — controlled by video fader in text overlay mode (D-11)
- `var recordingDuration: TimeInterval = 0` — elapsed timer
- `var cameraPermissionDenied = false`
- `private var controlsHideTask: Task<Void, Never>?`
- `private var durationTimer: Task<Void, Never>?`

**Computed properties:**
- `var hasTranscript: Bool { content.transcript != nil }` — gates Text segment in ViewModeControl
- `var isTextOnlyContent: Bool { content.contentType == .text }` — determines full-screen teleprompter vs video compositor
- `var effectiveVideoBlend: Float` — if `activeViewMode.faderControlsTextOpacity`, returns 0 (show reference only); otherwise returns `videoBlend`. The video fader value itself is always bound to `videoBlend`, but CompositorView receives `effectiveVideoBlend`.
- `var audioFaderVisible: Bool { content.contentType != .text }` — hide audio fader for text-only content (Open Question 2 from RESEARCH.md)
- `var formattedDuration: String` — formats `recordingDuration` as "MM:SS"

**Init:**
```swift
init(content: ReferenceContent) {
    self.content = content
    super.init()
}
```

**Methods:**

`func setup() async`:
- Check camera/mic authorization: `AVCaptureDevice.authorizationStatus(for: .video)`. If `.notDetermined`, request via `await AVCaptureDevice.requestAccess(for: .video)`. If denied, set `cameraPermissionDenied = true` and return.
- Same for `.audio`.
- Call `captureEngine.configure()`
- Call `captureEngine.startSession()`
- If content is video or audio: `playbackEngine.load(url: FileVault.url(for: content.filename!))` — load reference content
- Set initial audio: `playbackEngine.volume = 1.0` (audioBlend 0.0 = full reference)
- Call `FileVault.cleanupOldRecordings()` to prune temp files from prior sessions

`func toggleRecording()`:
- If not recording:
  - `syncTimestamp = captureEngine.syncTimestamp()` (REC-05: CACurrentMediaTime before async)
  - `let filename = "\(UUID().uuidString).mov"`
  - `let outputURL = FileVault.recordingURL(filename: filename)`
  - `captureEngine.startRecording(to: outputURL, delegate: self)`
  - If content is video or audio: `playbackEngine.seek(to: .zero)` then `playbackEngine.play()`
  - `isRecording = true`
  - `teleprompterScrolling = true` (D-13)
  - `recordingDuration = 0`
  - Start duration timer: `durationTimer = Task { while !Task.isCancelled { try? await Task.sleep(for: .seconds(1)); recordingDuration += 1 } }`
  - `scheduleControlsHide()` (D-04)
- If recording:
  - `captureEngine.stopRecording()`
  - `playbackEngine.pause()`
  - `isRecording = false`
  - `teleprompterScrolling = false`
  - `durationTimer?.cancel()`
  - `controlsVisible = true`

`func updateAudioBlend()`:
- Called when `audioBlend` changes (use `didSet` or call explicitly)
- `playbackEngine.volume = 1.0 - audioBlend` — left (0.0) = full reference volume, right (1.0) = no reference (FADER-02)

`func updateVideoBlendForMode()`:
- Called when `activeViewMode` changes
- If `.reference`: set compositor to show reference only (playerLayer opacity 1, previewLayer opacity 0 → `effectiveVideoBlend` returns 0)
- If `.camera`: set to camera only (previewLayer opacity 1 → `effectiveVideoBlend` returns 1)
- If `.blend`: use `videoBlend` as-is from fader
- If `.textOverlay`: show reference, `textOverlayOpacity = videoBlend` (D-11)

`private func scheduleControlsHide()` — per Pattern 7:
- `controlsHideTask?.cancel()`
- `controlsHideTask = Task { try? await Task.sleep(for: .seconds(3)); guard !Task.isCancelled, !isFaderDragging else { return }; withAnimation(.easeOut(duration: 0.4)) { controlsVisible = false } }`

`func showControls()`:
- `withAnimation(.easeIn(duration: 0.2)) { controlsVisible = true }`
- If `isRecording { scheduleControlsHide() }`

**AVCaptureFileOutputRecordingDelegate** — per RESEARCH.md code example:
- `nonisolated func fileOutput(_:didFinishRecordingTo outputFileURL:from:error:)`:
  - `Task { @MainActor in self.lastRecordingURL = outputFileURL; self.isRecording = false }`
- `nonisolated func fileOutput(_:didStartRecordingTo:from:)`:
  - `Task { @MainActor in self.isRecording = true }`

**Cleanup:**
- `func teardown()` — `captureEngine.stopSession()`, `playbackEngine.pause()`, cancel tasks
